# Starting with venv :
```
  python -m venv venvName
```
### activate venv :
```
  venvName\Scripts\activate
```
### Install Django :
```
pip install django
```
### Create a Project : 
```
django-admin startproject projName
```
### Create an App :
```
python manage.py startapp appName
```
### Run the server : 
```
python manage.py runserver
```
